## bash 命令重命名

* fa: 在当前目录下**所有文件**搜索指定关键字
* fs: 在当前目录下**所有源文件包括后缀为 .c .h .S .s**搜索指定关键字
* fc: 在当前目录下**所有 C 文件**搜索指定关键字
* fh: 在当前目录下**所有 .h 文件**搜索指定关键字
* wcl: 统计当前目录下**所有 C 文件**代码行数
* cls: 强制清屏, 不同于 clear 命令
